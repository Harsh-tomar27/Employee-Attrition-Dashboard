# HR-Analytics-Attrition-Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes employee attrition and workforce trends to identify the key factors influencing employee turnover. The dashboard helps HR teams make data-driven decisions to improve employee retention.

## 🎯 Objectives
- Analyze employee attrition patterns
- Identify factors affecting employee turnover
- Monitor key HR metrics
- Support employee retention strategies

## 📊 Dashboard Features
- Employee Attrition Rate
- Employee Count
- Department-wise Analysis
- Job Role Analysis
- Salary Slab Analysis
- Age Group Analysis
- Overtime Analysis
- Years at Company Analysis
- Interactive Filters (Slicers)

## 💡 Key Insights
- Employees with lower salaries showed higher attrition.
- Overtime employees had a higher turnover rate.
- Employees with shorter tenure were more likely to leave.
- Promotion opportunities influenced employee retention.
